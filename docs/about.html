About

v0.1.0

This is a simple application that I wrote to help me organize the attendance reports from Zoom meetings. The context is one or more a sychronous online college courses. That is, one or more recurring meetings are set up and a a dozen or more students participate in each over a semester's period of time.

The problem being solved is that the downloaded Zoom attendance reports are simple spreadhsheets (in CSV format) that organize the data by meeting instance, listing all the participants with thier join and leave times. Whie this is useful to get an idea of individual meeting instances, it doesn't help if you are interested in a specific atendee's participation over the course of the semester. This simple application just makes it easier to view attendance records, with a student focus.

It is 100% HTML and Javascript, hosted on GitHub pages with the source code avaiable as a public application under teh MIT licesnse. It does not require a server or separate executable engine, and runs completely inside a compatible browser. This means several things;

  • No data is ever transmitted over the network. All downloaded is stored locally in the browser's IndexedDB storage.
  • When Zoom CSV files are "uploaded" into the browser application, it is only persistent if you use the same browser (e.g., Chrome or Firefox) on teh same computer each time to use the application.
  • This application uses several JavaScript libraries from the CDN JSDeliver. These include:
    • JQuery
    • PapaParse
    • Luxon
    • IDB
    • Bootstrap
    • FullCalendar
    Unless you manually download all these components and this application's files, and edit them, you will need to use this application while connected to the Internet.

Hope you find this app useful.
